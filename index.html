Bible Reading Plan

About

Reading the Bible cover to cover from Genesis to Revelation is great, but has the downside that the order of the books is in a way that it takes the reader a long time until he reaches the New Testament, which can be demotivating. There are various solutions e.g. some read the Old and New Testament in parallel. However, I haven't felt happy with that, so I created this small program to automatically generate a pathway through the Bible to read it cover to cover. This program randomizes the order of the Bible books while ensuring that the Old Testament and New Testament get read in a balanced way.
